Module: lib/auth

Authentication and authorization utilities for the BashaChai API. This module handles user authentication using both Firebase ID tokens and custom JWT tokens. It also provides role-based access control, rate limiting, and user management functions.

(async, static) verifyApiToken(req) → {Promise.<Object>|string|string|string|string|string|boolean|string}
